# Landing Page JC Borges

Esta página está pronta para ser publicada no GitHub Pages.

## Como publicar

1. Envie o projeto para o repositório no GitHub.
2. Acesse as configurações do repositório.
3. Vá em "Pages".
4. Em "Build and deployment", selecione "GitHub Actions".
5. Faça push na branch `main` para disparar o deploy.

## Arquivos importantes

- `index.html` — página principal
- `styles.css` — estilos do site
- `.github/workflows/deploy-pages.yml` — workflow de publicação
- `.nojekyll` — evita tratamento do site como projeto Jekyll
